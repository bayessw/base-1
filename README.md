# My Solidity Token Project

Bu proje, Ethereum ağında çalışan standart bir **ERC-20** akıllı kontratıdır.

## Özellikler
* **Token Adı:** Github Gemini Token
* **Sembol:** GGT
* **Standart:** ERC-20
* **Güvenlik:** OpenZeppelin kütüphanesi kullanılarak yazılmıştır.

1. Repo'yu klonlayın.
